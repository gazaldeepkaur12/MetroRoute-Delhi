# MetroRoute Delhi

MetroRoute Delhi is a community-built web platform that helps commuters quickly check temporary Delhi Metro disruptions and find suitable alternate stations.

## Version 1 Features

- Search any Delhi Metro station
- View station status (Open / Closed / Limited)
- Suggested alternate stations
- Last updated timestamp
- Mobile-friendly interface

## Tech Stack

- JavaScript
- JSON
